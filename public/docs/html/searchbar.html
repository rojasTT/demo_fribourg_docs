Barre de recherche#

In diesem Textfeld können sowohl Datenebenen gesucht, als auch Koordinaten oder wichtige amtliche Einteilungen eingeben werden: Gemeinden, Adressen, Postadressen und Postleitzahlen, sowie Parzellen (Liegenschaften), dauernde und selbständige Rechte, Ortsnamen, Eidgenössische Gebäude ID (EGID) und Baubewilligungen (DATEC-Nummer).

Das auf der rechten Seite des Suchfelds angezeigtes Symbol gibt Ihnen einen Hinweis auf die Situation:

  • Eingabe gewartet : Es ist möglich eine neue Suche zu starten.

  • Zu wenig Zeichen* : Mindestens drei Buchstaben/Ziffern müssen eingegeben werden.

  • In Verarbeitung : Eine Suche kann einige Sekunden brauchen.

  • Keine Ergebnisse : Die Suche hat zu keinem Ergebnis geführt.

Es kann notwendig sein, die Resultate zu verfeinern, indem die Suche angepasst wird.

Ortsuche#

Ein Klick auf ein Ergebnis zentriert die Karte auf die Auswahl.

Suche nach Datenebenen#

Die Datenebenen erscheinen unter der Rubrik „Daten“. Ein Klick auf das Ergebnis fügt die Datenebe der Karte hinzu (unter der Rubrik „Ausgewählte Ebenen“)

Wenn die Ergebnisliste zu lange ist, wird die Anzahl Antworten unter der Rubrik „Daten“ angezeigt. Man kann auf die Rubrik klicken, um sämtliche Ergebnisse anzuzeigen.

Koordinatensuche#

Koordinaten können im Suchfeld eingegeben werden. Sobald X- und Y-Koordinaten komplett sind, wird die Karte auf den entsprechenden Punkt zentriert (gelbes Kreuz) und ein Fenster erscheint mit verschiedenen Optionen.

Es können sowohl Schweizer Koordinaten (System LV03 oder LV95; z.B. „578053 183596“) als auch geografische Koordinaten (z.B. „46.8 7.15“) gesucht werden. Der Punkt „.“ muss als Trennzeichen zu Dezimalstellen (anstelle des Kommas) verwendet werden.
